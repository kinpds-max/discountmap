# 전국허브 (National Discount Map Hub)

Steve Jobs Sensibility Premium Discount Platform.

## Features
- **Apple Design System**: Minimalist, Glassmorphism, Space Gray/Silver palette.
- **Extreme Discounts**: Only products with significant (30%+) discounts are highlighted.
- **Offline/Online**: Toggle between local map search and curated online deals.
- **Kakao Maps**: Integrated with custom markers for offline discovery.

## Getting Started
1. Open `index.html` in your browser.
2. Replace `YOUR_KAKAO_API_KEY` in `index.html` with your actual Kakao Developer API key.
3. Manage your discount data in `data.js`.

## Assets
- `gold_discount_seal`: A premium seal for verified extreme discounts.
- `national_hub_hero`: A high-end splash background.

Designed for those who seek the best value with the best style.
